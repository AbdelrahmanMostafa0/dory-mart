# DoryMart

## Overview
**DoryMart** is an e-commerce platform inspired by the vibrant and playful spirit of Dory from *Finding Nemo*. It aims to provide users with a seamless shopping experience, featuring an intuitive interface and fun design elements.

## Features
- **Product Listings**: Browse products with detailed descriptions, pricing, and images.
- **Search and Filtering**: Easily find products using search functionality and filters.
- **Shopping Cart**: Add products to your cart and manage them before checkout.

## Technologies Used
- **State Management**: Utilizes Redux for efficient state management across the application.
- **Animations**: Implements GSAP to enhance user experience with playful and engaging animations.
- **Performance & SEO**: Uses server-side rendering (SSR) for faster load times and improved SEO.
- **Persistent Cart**: Stores cart data in local storage, ensuring users don’t lose their selections when they refresh the page.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AbdelrahmanMostafa0/dory-mart.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dory-mart
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open the app in your browser at [http://localhost:3000](http://localhost:3000).
